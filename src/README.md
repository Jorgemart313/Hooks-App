# Hooks-App

https://hooks-app-pied.vercel.app/

¡Bienvenido a mi web de aprendizaje con hooks! Aquí puedes probar cualquiera de ellos y divertirte un poco con estas tareitas de mi curso.

Este repositorio contiene una colección de ejemplos interactivos de varios React Hooks (y algunos personalizados) para demostrar su uso y funcionalidad en diferentes escenarios. La aplicación está construida con React y TypeScript, estilizada con Tailwind CSS.

## Características

- **`useState`**: Demostraciones básicas del manejo del estado.
- **`useEffect`**: Ejemplos de efectos secundarios, como temporizadores y llamadas a API.
- **`useRef`**: Uso de referencias para acceder a elementos DOM o valores mutables.
- **Custom Hooks**: Creación y uso de hooks personalizados para lógica reutilizable.
- **Navegación sencilla**: Una página principal con botones para explorar cada demo individualmente.

## Tecnologías Usadas

- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **TypeScript**: Lenguaje que agrega tipado estático a JavaScript.
- **Tailwind CSS**: Framework CSS de utilidad para estilizado rápido y responsivo.
- **Vite**: Herramienta de compilación rápida para desarrollo web.

## Demos Disponibles

La página principal (`HooksShowcase`) te permitirá navegar por los siguientes ejemplos:

- **Home (HooksApp)**: Página de bienvenida general.
- **useState (Semáforo Simple)**: Demostración básica de `useState` para cambiar el estado de un semáforo manualmente.
- **useEffect (Semáforo con Timer)**: Un semáforo que cambia automáticamente de color usando `useEffect` para gestionar un temporizador.
- **Custom Hook (Semáforo)**: El mismo semáforo con temporizador, pero encapsulando toda su lógica en un hook personalizado `useTrafficLight`.
- **useCounter (Pokémon)**: Un contador que te permite navegar por diferentes Pokémon, usando un custom hook `useCounter` y realizando peticiones a una API con `useEffect`.
- **useRef (Focus Input)**: Ejemplo de cómo usar `useRef` para enfocar un elemento de entrada de texto al hacer clic en un botón.

¡Diviértete explorando los hooks!
